Search Engine

In this project, I built a simple search engine. I first wrote a web crawler to fetch docuemnts from the neu.edu domain and created a forward index of the documents. Next, I wrote another program to convert this into an inverted index and stored the index in a file, with delta encoding. Further, I implemented a simple search utility to fetch documents based on query keywords and ranked the results using scoring functions like TF, TF-IDF, BM-25 and JM Smoothing.

Twitter Data Analysis

Twitter data analysis provides some valuable insights to help businesses. I worked in a team of 4 (including me) to understand the sentiment behind the tweet. For this we wanted to explore the difference in sentiment(opinion) in a few cities across the United States, for which we collected data by tracking a few keywords and geolocations. Then we used MapReduce to identify tweets with same hashtags and classified each tweet using the Stanford NLP library. This gave us an understanding of how people from different cities tweet using the same hashtag(positive/ negative/ neutral).
